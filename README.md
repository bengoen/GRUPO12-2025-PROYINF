## Proyecto base identificado

Durante el semestre 2026 el equipo continuará trabajando sobre el proyecto base desarrollado en el semestre 2025-2 en el repositorio `GRUPO12-2025-PROYINF`.

El sistema heredado corresponde a **Tu Préstamo Digital**, una aplicación web para la gestión de préstamos personales digitales, construida principalmente con `Node.js`, `Express`, `EJS`, `PostgreSQL` y `Docker`, contenida en la carpeta `mi-proyecto-node-docker`.

## Alcance heredado identificable en el repositorio

- `HU001`: simulación de préstamos y registro de solicitudes.
- `HU002`: monitoreo del estado de las solicitudes.
- `HU003`: contrato, firma digital simulada y visualización de préstamos activos.
- `HU004`: registro e inicio de sesión básico del solicitante.
- `HU006`: pago de cuotas con Transbank en ambiente de integración.
